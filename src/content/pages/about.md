# About GNNL TV

Welcome to GNNL TV! This is a simple page template that you can use for any static content page on your site.

## Editing This Page

To edit this page, simply modify the content in `src/content/pages/about.md`. You can add more sections, paragraphs, images, or any other content you need.

## Creating More Pages

To create additional simple pages like this one:

1. Create a new `.md` file in the `src/content/pages/` directory
2. Add frontmatter with `title` and `description`
3. Write your content in Markdown
4. Create a corresponding `.astro` page in `src/pages/` that loads this content
